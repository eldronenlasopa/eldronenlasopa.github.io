# Card

Surface container: white, rounded (`--radius-lg`), 1px subtle border, soft shadow. `interactive` adds a hover lift; `accent` adds a top orange bar for feature cards.

```jsx
<Card elevation="md" interactive accent>
  <h3>Aplicaciones web</h3>
  <p>Del prototipo a producción.</p>
</Card>
```

Props: `elevation` (none·sm·md·lg), `interactive`, `accent`, `padding`.

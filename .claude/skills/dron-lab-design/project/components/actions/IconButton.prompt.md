# IconButton

Circular button for a single icon glyph — use in toolbars, cards, and nav. Requires a `label` for accessibility.

```jsx
<IconButton label="Cerrar" variant="ghost">✕</IconButton>
<IconButton label="Menú" variant="outline">☰</IconButton>
```

Variants: `solid` · `ghost` · `outline`. Sizes: `sm` (34px) · `md` (42px) · `lg` (50px).
